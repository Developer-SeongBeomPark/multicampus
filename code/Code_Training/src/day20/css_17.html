display : inline ; contents가 있어야 보여진다. 콘텐츠 크기로 보여진다. 한줄에 여러개가 보여진다. display : block ; 콘텐츠 크기와 상관없어 css의 width, height 사이즈로 보여준다. 한줄에 하나만 보여진다. display : inline-block ; 콘텐츠 크기와 상관없어 css의 width, height 사이즈로 보여준다. 한줄에 여러개가 보여진다. - position : static (default) - html에 정의된 순서대로 브라우저에 자연스럽게 보여줌 - position : relative - 나의 원래자리에서 상대적으로 이동 - position : absolute - 내 아이템이 담겨있는 상자에서 이동됨 - position : fixed - 내 아이템이 담겨있는 상자에서 벗어나 원도우 즉 page 상에서 이동된다 - position : sticky - 원래자리에 고정되어, 스크롤바를 이동해도 그자리에 그대로 있다.
